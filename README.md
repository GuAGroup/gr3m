# GR3M Protocol 🛡️

**GR3M** — это высокопроизводительный асинхронный сетевой туннель, разработанный для обхода глубокого анализа трафика (DPI) с использованием динамической маскировки под стандартные протоколы (WebSocket/HTTP). Проект ориентирован на максимальную скрытность, обеспечивая защиту данных и метаданных пользователя.

## ✨ Ключевые особенности

* **Perfect Forward Secrecy (PFS):** Каждая сессия использует уникальные эфемерные ключи X25519. Старый трафик невозможно расшифровать даже при компрометации долгосрочных секретов.
* **Защита от Replay-атак:** Встроенная система Nonce (Sequence Numbers) полностью исключает возможность повторной отправки перехваченных пакетов злоумышленником.
* **Smart Padding:** Динамическое заполнение пакетов случайными данными (Length Hiding) для предотвращения анализа трафика по размеру фреймов.
* **Remote DNS:** Резолвинг доменных имен происходит на стороне сервера. Это предотвращает утечки DNS и позволяет обходить блокировки на уровне провайдера.
* **Hysteria Masking:** Система "приманки" (Decoy). При обнаружении некорректного подключения или попытке сканирования, сервер маскируется под легитимный веб-ресурс.
* **Latency-based Peer Selection:** Клиент автоматически тестирует доступные узлы и подключается к серверу с минимальной задержкой.

## 🚀 Быстрый старт

### Требования

* Go 1.22+ (Рекомендуется 1.26+)
* Docker & Buildx (для контейнеризации)

### Сборка

```bash
# Подготовка зависимостей
go mod tidy

# Сборка образа через Docker с использованием BuildKit
DOCKER_BUILDKIT=1 docker build -t gr3m-app .

```

### Настройка (`config.json`)

```json
{
  "mode": "client",
  "listen_addr": "0.0.0.0:8080",
  "peers": [
    {"name": "Main-Server", "addr": "your-vps-ip:8080"}
  ],
  "socks_addr": "127.0.0.1:1080",
  "decoy_url": "wikipedia.org:80"
}

```

## 🛠️ Использование

### Запуск Сервера (на VPS)

```bash
docker run -d \
  --name gr3m-server \
  -p 8080:8080 \
  --restart always \
  gr3m-app

```

### Запуск Клиента (Локально)

```bash
docker run -d \
  --name gr3m-client \
  -p 1080:1080 \
  gr3m-app

```

*Настройте вашу систему или браузер на использование SOCKS5 прокси: `127.0.0.1:1080`.*

## 🏗️ Архитектура проекта

* `/protocol`: Реализация криптографического слоя (ChaCha20-Poly1305), компрессии и защиты от Replay.
* `/core`: Логика хендшейка, обмена ключами и управления конфигурацией.
* `/client`: SOCKS5-прокси сервер, асинхронный мост и выбор пиров.
* `/server`: Релей-движок, динамическое проксирование и система маскировки.
